# MARTOVER 用户指南

## 🚀 欢迎来到TOVER联盟！

欢迎，第101号成员！您即将踏上一段激动人心的火星农业探索之旅。本指南将帮助您快速上手，成为一名合格的火星农业工程师。

## 📖 快速开始

### 第一步：阅读信封

当您首次进入应用时，会看到三封来自TOVER联盟的信件：

1. **欢迎信** - 了解您的身份和TOVER联盟
2. **任务说明** - 学习火星土豆培育的重要性
3. **出发准备** - 明确您的具体任务目标

💡 **提示**: 
- 点击"下一页"翻阅信件
- 背景音乐会随着阅读进程变化
- 仔细阅读每封信，这些信息对您的火星之旅至关重要！

### 第二步：探索火星

阅读完信件后，您将穿越云层降落到火星表面。

**操作方式：**
- 🖱️ **鼠标拖拽** - 旋转火星地球仪
- 🔍 **滚轮缩放** - 放大/缩小视图
- 👆 **点击标记** - 选择陨石坑查看详情

### 第三步：选择农业基地

点击火星表面的陨石坑标记，右侧会弹出详细信息面板：

```
┌─────────────────────┐
│ ID: CRATER_001  ✕   │ ← 基本信息
│ 位置: 45.2°N, 12.1°E│
│ 直径: 15.6 km       │
├─────────────────────┤
│      🎯 图例        │ ← 可视化
│   [陨石坑形状图]    │
├─────────────────────┤
│ 层数: 3             │ ← 详细数据
│ 溅射物形态: SL, BL  │
│ 坑内形态: Cpx       │
│ 退化程度: ▓▓▓░░    │
│                     │
│  [选择此陨石坑]     │
└─────────────────────┘
```

**数据解读：**

| 指标 | 说明 |
|------|------|
| 直径 | 陨石坑大小（公里） |
| 层数 | 溅射物层次结构 |
| 溅射物形态 | SL(单层)、BL(双层)、ML(多层) |
| 坑内形态 | Cpx(复杂)、Smp(简单)、Bsn(盆地) |

**退化程度指示：**
- 🟢 新鲜 - 保存完好，适合建设
- 🟡 轻微退化 - 基本完整，可以使用
- 🟠 中度退化 - 部分损坏，需要评估
- 🔴 严重退化 - 不建议选择

### 第四步：土豆培育

选择陨石坑后，您将进入土豆培育界面：

- **3D土豆星球** - 创新的品种展示方式
- **品种属性** - 详细的适应性数据
- **培育建议** - 基于火星环境的种植指导

### 第五步：自动化生产

完成土豆培育后，进入自动化生产建设阶段，查看火星本土化生产系统。

## 🎵 音频体验

应用包含沉浸式配乐：
- **背景音乐** - 全程循环播放的宇宙氛围音乐
- **信封音乐** - 阅读信件时的特殊配乐
- **降落音乐** - 穿越云层降落火星时的音效

## ❓ 常见问题

**Q: 为什么没有声音？**
A: 浏览器安全策略要求用户交互后才能播放音频。请点击页面任意位置或按钮。

**Q: 3D场景加载很慢？**
A: 首次加载需要下载3D资源，请耐心等待。建议使用现代浏览器（Chrome、Firefox、Edge）。

**Q: 如何重新开始？**
A: 刷新页面即可重新开始体验。
